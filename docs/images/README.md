## Contributing Images

If you'd like to contribute screenshots:
- Take clear photos or screenshots from RadioMaster TX16S
- Submit via GitHub issue or pull request
- Name descriptively (e.g., `widget-selection.png`)

Images are optional - the installation guide works without them.
