## Image Guidelines

- **Format**: PNG preferred for screenshots, JPG acceptable for photos
- **Resolution**: Minimum 480x272 (TX16S native resolution), higher is better
- **Quality**: Clear, readable text and UI elements
- **Consistency**: Try to use images from the same radio model (TX16S) when possible
- **Alternatives**: If TX16S screenshots aren't available, similar EdgeTX color screen radios (X10, X12S) are acceptable

## Creating Placeholder Images

Until actual screenshots are available, you can:

1. Use similar images from EdgeTX documentation
2. Use screenshots from other widget installation guides
3. Take photos of the actual TX16S screen during installation
4. Use screen capture from EdgeTX Companion simulator

## Image Attribution

If using images from other sources:
- Credit the source in the image filename or alt text
- Ensure images are licensed for reuse (CC, MIT, GPL, or public domain)
- EdgeTX official documentation images can be used under their license

## Contributing Images

To contribute actual screenshots:
1. Follow the naming convention: `##-descriptive-name.png`
2. Ensure images are clear and well-lit (for photos)
3. Crop to show relevant UI elements
4. Add the image to this directory
5. Update the installation guide if needed

**Note**: The installation guide is designed to work with or without images. Alt text provides descriptions for accessibility.
