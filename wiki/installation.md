## copy to tx
Go to releases and download the latest version of Rfsuite edgetx

![Image](https://github.com/user-attachments/assets/845800eb-362c-43c8-a150-8f30c1105272)


Extract the zip file and copy the Script and widgets folder tot the sd-card of the Edgetx radio.

![Image](https://github.com/user-attachments/assets/15275e1a-68da-4f30-90ca-a9ca4bd7cc04)




## Radio setup

Start the radio and go to System -> Tools. Click on the _RF2 Touch script. The first time it will compile all the files. 

![Image](https://github.com/user-attachments/assets/bef95723-fe3f-4d03-b042-313e29ea72a0)


When the compiling is finished it will return to the tools page. Click then again on _RF2Touch and it should open like this:

![Image](https://github.com/user-attachments/assets/f6a25f88-552e-4add-a40f-41fdfef4a594)


## core widgets settings 
Go to homescreen

![Image](https://github.com/user-attachments/assets/f65d1bcb-8d5b-4d65-8655-953aa06d4786)

Press the roller and click on Screens Settings

![Image](https://github.com/user-attachments/assets/64930cc8-daec-4651-a8d5-6a4539cd924e)

Click on the top bar and click on Setup widgets

![Image](https://github.com/user-attachments/assets/7420f9a5-3746-44c2-8b1c-90e7f8c3c11f)

Click on the top bar at a free slot for a widget. In this case on the right side. Select here rf2_widget. This widget will take care of the connection to the FC.

![Image](https://github.com/user-attachments/assets/baab33f5-8dd1-4582-9c30-d10f9059eb5b)

## widget setting
At the top bar you can also use a name widget e.g. rf2_name. This will display the name of the aircraft

![Image](https://github.com/user-attachments/assets/e0faaff8-3fc6-4f7b-a816-4f76bed9130c)

Click on return and go back to the screen settings. Click on screen 1 make it a full screen layout and disable the Flightmode, sliders, trims and mirror. Click on Setup widgets.

![Image](https://github.com/user-attachments/assets/34686e04-8e08-433f-94cc-1faad21dc133)

Click on the screen in the middle and select here rf2_dash in the list. 

![Image](https://github.com/user-attachments/assets/9305dac5-5c94-4a0a-b407-097ab64dd3c3)

Click multiple on return untill you are back in the screen. This will look like the picture below if you are connected with the heli.

![Image](https://github.com/user-attachments/assets/b9a1a654-f5e5-4b40-a373-c654c92f859b)
