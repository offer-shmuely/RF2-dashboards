# RF2-dashboards Wiki

This wiki contains documentation for the RF2-dashboards project, a suite of LUA scripts and widgets for EdgeTX radio systems running Rotorflight 2.

## Contents

- **[Home](Home.md)** - Overview and introduction to the RF2-dashboards suite
- **[Installation](installation.md)** - Step-by-step installation instructions
- **[RF_touch](RF_touch.md)** - Touch version of the Rotorflight setup script

### Widgets

- **[rf2_server](widget-rf2_server.md)** - Background communication widget (required for all other widgets)
- **[rf2_dashboard](widget-‐-rf2_dashboard.md)** - Main dashboard widget showing telemetry data
- **[rf2_blackbox](widget-‐-rf2_blackbox.md)** - Blackbox widget documentation
- **[rf2_name](widget-‐-rf2_name.md)** - Name widget for displaying aircraft name

## Quick Start

1. Follow the [Installation](installation.md) guide to set up the scripts on your radio
2. Configure the [rf2_server](widget-rf2_server.md) widget on your top bar
3. Add the [rf2_dashboard](widget-‐-rf2_dashboard.md) widget to your main screen
4. Start flying!
